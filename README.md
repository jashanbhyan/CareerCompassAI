# CareerCompass AI 🚀

An end-to-end Data Science project that collects live job market data using REST APIs, stores it in PostgreSQL, performs data cleaning, exploratory data analysis, statistical testing, machine learning, and visualizes insights through Power BI and Flask.

## 🚀 Features

- Live Job Data Collection
- PostgreSQL Database
- Data Cleaning
- Exploratory Data Analysis
- Statistical Analysis
- Salary Prediction
- Skill Demand Analysis
- Power BI Dashboard
- Flask Web Application

## 🛠 Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- PostgreSQL
- Scikit-Learn
- Flask
- Power BI
- Git & GitHub
